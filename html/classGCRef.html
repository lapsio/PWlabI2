Detailed Description

The GCRef class Class providing basic garbage collecting functionality similar to one provided by sharedptr.

User can get link which can be deleted later to inform that certain code won't use data anymore. Once all links get deleted class is deleting itself on its own. New class has 0 links thus it's locked. It can be manually unlocked, then deleted or it unlocks automatically after registering first link.
